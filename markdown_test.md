## Commandes terminal
| Commandes | Ce qu'elle fait | Exemple |
| --------- | --------------- | ------- |
| pwd | Permet de savoir où on se trouve | pwd |
| cd | Entrer dans un dossier | cd projet_operateur |
| touch | Créer un nouveau fichier | touch test.txt |
| mkdir | Créer un nouveau dossier | mkdir projet_operateur |
| ls | Lister les fichiers du répertoire actuel | ls |
